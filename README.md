## JSON File Format
```json
{
    "file1": {
        "filename": "file1",
        "times": {
            "start_1": "18:00",
            "end_1": "00:30",
            "start_2": "22:10",
            "end_2": "00:30"
        },
        "merge_clips": true
    },
    "file2": {
        "filename": "file2",
        "times": {
            "start_1": "8:00",
            "end_1": "00:30",
            "start_2": "2:10",
            "end_2": "00:30"
        },
        "merge_clips": false
    }
}
```
